# Component Library

The `stdlib/components` module contains a set of atomic and molecular UI components built with Tailwind CSS.

## Base Components
- `BaseButton`: Highly configurable button with multiple variants (Primary, Secondary, Outline, Ghost, Error).
- `BaseInput`: Accessible input field with label and validation support.
- `BaseCard`: Container for grouping related content with optional titles.
- `BaseBadge`: Small status or label indicators with custom coloring.

## Complex Components
- `Tabs`: A system for switching between different content panes.
- `Modal`: Accessible dialog overlay for confirmations and forms.
- `NotificationToast`: Global notification system for user feedback.
- `StatsCard`: KPI display card with trend indicators.
- `Showcase`: Utility for displaying component previews alongside their source code.
- `CodeBlock`: Syntax-highlighted code viewer for documentation and demos.

## Design Tokens
The library relies on Tailwind CSS for styling. Theme support is integrated via the `GlobalStore`, allowing seamless switching between light and dark modes.
